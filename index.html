Welcome to Swarthmore College's Bach Chorale Workshop! This site is designed for anyone interested in learning to analyze and transcribe multiple voices sounding simultaneously. The instrument assignment and volume of each voice are customizable so that you can concentrate on particular elements in the texture and work up to being able to transcribe all four voices at once.

To get started, select one of over 400 chorales from the drop-down menu. Enter practice mode at any time to make it more challenging! For more information, see the about page.
