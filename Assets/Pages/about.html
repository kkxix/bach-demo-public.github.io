Instructions

To use Bach Chorale Workshop, navigate to the home page and select or search for one of over 400 chorales. The first number of each chorale refers to the BWV number of the cantata from which it is excerpted, while the second number (for BWV 1-250) refers to the movement of the cantata in which you'll find the chorale.

After loading, you will see a score of the piece to the right and the sound controls to the left. Using the sliding bars, you can go to any second in the music and set the tempo anywhere between 50 and 200 bpm. There are also separate instrument and volume controls for each voice of the chorale.

For more of a challenge, toggle the practice mode button at the top. This will hide the score and limit your volume-changing abilities so that you can only change the volume of the soprano/alto parts or tenor/bass parts at the same time. You can still set separate instruments for each voice. Please note that highlighted chorales are not available in practice mode at this time.

Project Background

This project was designed to enable ear training students to have richer opportunities to practice hearing multiple voices at once. Previously, students had to choose between taking dictation of chorales from commercial recordings (where the acoustic circumstances and blending made the prospect of transcribing all four voices a very advanced task indeed) or having a friend or teacher play the chorales on the piano (where it is nearly impossible to distinguish crossing inner voices). While the Bach chorales have long been available in MIDI format, manipulating the timbre and balance of each voice was a labor intensive process and put many steps between the student and the exercise that would be most helpful to them. It is our hope that this site fills the gap in the process and can enable students to work up to transcribing all four voices of a chorale.

Code

This site hosted on GitHub Pages. Using a static collection of Bach chorales in MIDI and MUSICXML format and libraries that convert music events into data types that can be interpreted and adjsuted by your web browser's audio context, we created this simple player. Please visit our GitHub page to see the source code and the credits page for more information about the JavaScript libraries we used to build this tool.
